# 全息分光镜应用 - Holographic Spectroscope Apps

适用于 2.5cm x 2.5cm x 2.5cm 分光镜的全息投影网页应用集合。

## ✨ 功能场景

### 🕐 全息时钟
- 青色霓虹发光效果
- 秒数翻牌动画
- 实时日期显示

### 💖 3D 旋转心形
- 粉色发光心形
- 360° 持续旋转
- 心跳缩放效果

### 🎆 烟花效果
- 多彩粒子爆炸
- 随机发射轨迹
- 拖尾渐隐效果

## 🎮 操作方式

| 操作 | 功能 |
|------|------|
| 点击屏幕底部 | 显示场景切换按钮 |
| 3秒无操作 | 按钮自动隐藏 |
| 双击屏幕 | 切换全屏模式 |

## 🚀 使用方法

### 本地运行
```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8080

# 然后访问
http://localhost:8080
```

### 手机使用
1. 确保手机和电脑在同一网络
2. 在手机浏览器访问 `http://电脑IP:8080`
3. 点击底部显示切换按钮
4. 双击屏幕进入全屏模式
5. 将分光镜放在屏幕中央的方框位置

## 💡 最佳效果建议

- 屏幕亮度调到最高
- 在暗环境下使用
- 分光镜对齐屏幕中央的边框
- 使用全屏模式
