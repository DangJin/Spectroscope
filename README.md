# 全息分光镜应用 - Holographic Spectroscope Apps

适用于 2.5cm x 2.5cm x 2.5cm 分光镜的全息投影网页应用集合。

## 🕐 全息时钟 (Holographic Clock)

一个具有科技感的全息时钟，通过分光镜可呈现 3D 悬浮效果。

### 功能特性

- 纯黑背景（全息投影必需）
- 垂直翻转显示（适配分光镜反射）
- 青色霓虹发光效果
- 扫描线动画增强科技感
- 实时时间和日期显示
- 防止屏幕休眠
- 双击全屏模式

### 使用方法

1. **本地运行**
   ```bash
   # 使用 Python 启动本地服务器
   python3 -m http.server 8080
   
   # 然后访问
   http://localhost:8080
   ```

2. **手机使用**
   - 确保手机和电脑在同一网络
   - 在手机浏览器访问 `http://电脑IP:8080`
   - 双击屏幕进入全屏模式
   - 将分光镜放在手机屏幕上方

### 最佳效果建议

- 屏幕亮度调到最高
- 在暗环境下使用
- 分光镜放置在屏幕中央
- 使用全屏模式
